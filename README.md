# LiaoBa
一款基于融云SDK的社交APP，开发者需要在IMLib中替换成自己的appkey，然后在Constants文件中配置用户信息即可运行
<br>发送自定义消息是通过融云控制台发送。<br>
例如：<br>
fromUserId：001<br>
toUserId：002<br>
objectName：自定义<br>
            app:RedPkgMsg<br>
content：{"title":"恭喜发财","status":"500.00","type":"微信红包"}<br> 
pushData：这是pushData<br>
pushContent：这是pushContent<br>
